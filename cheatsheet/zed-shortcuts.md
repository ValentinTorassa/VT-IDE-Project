# Zed Shortcuts — VT IDE Project

## AI Commands
| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+A` | Toggle AI Assistant panel |
| `Cmd+Shift+I` | Inline AI assist |
| `Cmd+K` | Inline assist on selection |

## Navigation
| Shortcut | Action |
|----------|--------|
| `Cmd+P` | File finder |
| `Cmd+Shift+F` | Project-wide search |
| `Cmd+Shift+O` | Symbol finder |
| `Cmd+G` | Go to line |
| `Ctrl+Tab` | Switch between open files |

## Editing
| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+D` | Duplicate line down |
| `Cmd+Shift+K` | Delete line |
| `Alt+Up/Down` | Move line up/down |
| `Cmd+D` | Select next occurrence |
| `Cmd+/` | Toggle comment |
| `Cmd+Shift+[/]` | Fold/unfold code |

## Git
| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+G` | Toggle git panel |
| `Ctrl+\`` | Toggle terminal |

## Terminal Scripts (VT Tools)
| Command | Action |
|---------|--------|
| `vt-pr` | List/summarize PRs with AI |
| `vt-diff` | Explain current diff with AI |
| `vt-review` | AI code review on changes |
| `vt-conflict` | AI merge conflict resolver |
| `vt-commit` | AI commit message generator |

## Zed Built-in
| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+P` | Command palette |
| `Cmd+,` | Settings |
| `Cmd+K Cmd+S` | Keyboard shortcuts |
| `Cmd+\\` | Toggle sidebar |
| `Cmd+J` | Toggle bottom panel |
